# 📊 Trader Behavior Intelligence using Market Sentiment

## 📌 Overview

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using real trading data from Hyperliquid.

The goal is to uncover behavioral patterns and generate insights that can improve trading strategies.

---

## 🧠 Key Features

* Data cleaning and preprocessing of real-world datasets
* Time-based merging of sentiment and trading data
* Feature engineering (profitability, risk score, trade behavior)
* Trader segmentation using K-Means clustering
* Visualization of trader behavior patterns
* Insight-driven strategy recommendations

---

## 📊 Key Insights

* Traders perform better during **Fear market conditions**, indicating contrarian opportunities
* High leverage and risk do not consistently yield better returns
* A small group of traders consistently outperform others
* Emotional trading leads to poor performance

---

## 🧠 Trader Clusters

* **Cluster 0** → Low risk, low return traders
* **Cluster 1** → High risk, poor performers
* **Cluster 2** → High profit, optimized traders

---

## 🎯 Strategic Recommendations

* Use contrarian strategies during Fear markets
* Avoid excessive risk during Greed phases
* Focus on risk-adjusted returns
* Learn from high-performing trader behavior
